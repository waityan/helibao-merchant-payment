# 合利宝商户打款激活功能

> 创建时间：2026-05-19
> 页面路径：`/Users/waityan/Documents/视听云/财务工作集合/打款激活功能/index.html`
> 页面类型：PC后台，独立单页

---

## 需求

财务录入收款商户号后，系统自动向该商户打款（固定 0.01 元）。
付款商户号在后台配置，前端不展示。
需有批量打款能力和打款记录查看。

## 核心流程

- **打款操作**：弹窗触发，textarea 多行录入收款商户号（每行一个），备注选填
- 商户号格式：**11位**（字母+数字混合），需校验
- 打款金额：固定 **¥0.01**
- 确认后新增记录，状态为「打款中」，操作人固定为 admin
- **打款记录**：主页面表格展示，可筛选（收款商户号搜索 + 状态下拉 + 打款时间区间）

## 字段说明

| 列      | 来源               | 说明              |
|---------|--------------------|-------------------|
| ID      | 自增               | 主键              |
| 收款商户号 | 财务录入          | 11位，字母数字    |
| 付款商户 | 后台固定配置       | 前端隐藏，列表显示 |
| 金额   | 固定 ¥0.01        | —                 |
| 打款时间 | 系统自动生成       | 精确到分钟        |
| 状态   | 系统维护           | 打款中 / 成功 / 失败 |
| 操作人 | 取当前登录账号     | 目前 mock 为 admin |
| 备注   | 手动填写           | 选填，默认「商户激活打款」 |

## 技术实现

- 纯 HTML/CSS/JS 单文件原型
- Ant Design 风格自定义 CSS（品牌色 #1A1A2E，强调色 #1677FF）
- JS 状态：`mockData` 内存数组，每条新录入的 payload 插入数组首部
